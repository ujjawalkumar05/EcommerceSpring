# EcommerceSpring

A simple E-Commerce Backend Application built using Spring Boot. This project provides REST APIs for managing products, categories, users, carts, and orders in an online shopping system.

## Features

- User Management
- Product Management
- Category Management
- Shopping Cart Functionality
- Order Management
- RESTful APIs
- Database Integration
- Exception Handling

## Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Gradle
- REST API

## Project Architecture

The project follows a layered architecture:

- Controller Layer
- Service Layer
- Repository Layer
- Entity Layer

## Prerequisites

Before running the project, make sure you have:

- Java 17 or higher
- MySQL Database
- Gradle

## Installation

Clone the repository:

git clone https://github.com/ujjawalkumar05/EcommerceSpring.git

Move to the project directory:

cd EcommerceSpring

Configure your database credentials in the application properties file.

Run the application:

./gradlew bootRun

The application will start on:

http://localhost:8080
